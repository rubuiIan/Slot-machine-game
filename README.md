# 🎰 JavaScript Slot Machine (Node.js)

This is a simple Slot Machine game built using JavaScript and Node.js that runs in the terminal.  
The user can deposit money, place bets on lines, spin the slot machine, and win money based on matching symbols.

## 🚀 Features
- Deposit system
- Choose number of betting lines (1–3)
- Bet per line
- Random slot machine spins
- Winnings calculation
- Balance tracking
- Play again option
- Input validation

## 🧠 What I Learned
This project helped me practice:
- JavaScript functions
- Loops and conditionals
- Arrays and objects
- Random number generation
- Input validation
- Program flow and game logic
- Writing modular code

## 🛠️ Technologies Used
- JavaScript
- Node.js
- prompt-sync (for user input)
